# 🎯 Project Purpose

This project was built as part of the final hands-on benchmark in the **Hugging Face Agents Course**.

The goal was to:
- fetch benchmark questions
- process them with a custom agent
- generate clean final answers
- submit them through the official evaluation flow
- review the score and result

---

# 🛠️ What I Built

I built a Gradio-based Hugging Face Space application that:
- logs in with a Hugging Face account
- fetches benchmark questions
- runs custom agent logic in Python
- handles evaluation workflow
- submits results
- displays answer output and score
