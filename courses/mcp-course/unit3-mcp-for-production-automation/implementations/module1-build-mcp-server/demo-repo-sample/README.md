# Demo Repo Sample

This is the small local Git repository structure used during Module 1 testing with Claude Code.

It mirrors the safe demo branch used to let the PR Agent inspect file changes, recommend a PR template, and generate a cleaner pull request summary.

Files included:
- `app/auth.py`
- `docs/readme.md`
- `tests/test_auth.py`

In practice, this sample repo was initialized as a local Git repository, committed once on `main`, and then modified on a feature branch before attaching the MCP server inside Claude Code.
