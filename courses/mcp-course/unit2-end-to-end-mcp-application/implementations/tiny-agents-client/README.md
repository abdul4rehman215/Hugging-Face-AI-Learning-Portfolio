# Tiny Agents MCP Client

This folder contains the Tiny Agents configuration used in the final Unit 2 client exercise.

## What it demonstrates
- remote MCP connectivity through `mcp-remote`
- Tiny Agents loading the exposed MCP tool
- successful end-to-end invocation of the sentiment analysis tool

## Files
- `agent.json` — configuration template
- `package-notes.md` — install and run notes

## Important
The public version intentionally replaces the original server IP with `YOUR_SERVER_PRIVATE_IP`.
