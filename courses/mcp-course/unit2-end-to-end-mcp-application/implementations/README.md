# Implementations

This directory contains the practical files reconstructed from the Unit 2 hands-on work.

## Included parts
- `gradio-mcp-server/` — server-side sentiment analysis MCP app
- `client-configs/` — generic MCP config templates and shell env examples
- `continue-cli-client/` — Continue CLI configuration adapted for EC2/terminal use
- `gradio-mcp-client/` — Gradio app acting as an MCP client
- `tiny-agents-client/` — Tiny Agents configuration and notes

## Public repo note
These files are structured to be publishable:
- no API keys
- no private IPs hardcoded
- placeholders or environment variables are used where needed
