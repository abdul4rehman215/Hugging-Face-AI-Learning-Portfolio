# MCP Client Config Templates

These files capture the basic client-side config structures used during the Unit 2 practice.

## Included files
- `mcp.json` — generic MCP configuration example
- `config.json` — UI-style config example
- `server.env.example` — shell variables used while testing from the client EC2 instance

## Important
The repository version uses placeholders like `YOUR_SERVER_PRIVATE_IP` so the files are safe to publish.
Replace them with your own endpoint when testing.
