# 📘 Unit 1: Fundamentals of Large Language Models (LLMs)

## 📌 Overview
This unit introduces the core concepts behind Large Language Models (LLMs) and modern Natural Language Processing (NLP).

It focuses on understanding how transformer-based architectures work and how they power today’s AI systems.

---

## 🧠 Key Concepts Learned

### 🔹 1. What are LLMs?
- Definition and capabilities of Large Language Models
- Differences between traditional NLP and modern LLM approaches
- Real-world applications (chatbots, summarization, QA, etc.)

---

### 🔹 2. Transformer Architecture
- Why transformers replaced RNNs and LSTMs
- Self-attention mechanism
- Context understanding vs sequential processing

---

### 🔹 3. Hugging Face Ecosystem
- 🤗 Transformers library
- 🤗 Datasets and Tokenizers
- Hugging Face Hub (models & sharing)

---

### 🔹 4. Working with Pretrained Models
- Loading models from Hugging Face
- Running inference using pipelines
- Understanding outputs

---

### 🔹 5. NLP Tasks Covered
- Text Classification
- Named Entity Recognition (NER)
- Question Answering

---

### 🔹 6. Fine-Tuning Basics
- Concept of transfer learning
- Adapting models to custom tasks

---

### 🔹 7. Limitations of LLMs
- Bias in models
- Hallucinations
- Ethical considerations

---

## 🎯 Key Takeaways
- Strong conceptual understanding of transformers
- Ability to use pretrained models in real applications
- Foundation for advanced LLM topics and fine-tuning
