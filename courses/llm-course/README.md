# 🤖 Hugging Face LLM Course

This repository documents my learning journey through the Hugging Face Large Language Models (LLM) Course.

## 📌 About the Course
The Hugging Face LLM Course is a comprehensive program designed to build a strong foundation in modern Natural Language Processing (NLP) using transformer-based architectures.

It covers:
- Transformers & attention mechanisms
- Pretrained models and fine-tuning
- Real-world NLP tasks
- Hugging Face ecosystem (Transformers, Datasets, Hub)
